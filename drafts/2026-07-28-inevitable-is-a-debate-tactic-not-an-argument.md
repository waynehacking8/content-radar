There's a sharp essay called "LLM Inevitabilism" that names something I'd noticed but hadn't put a label on: a lot of AI discourse leans on inevitability as its main rhetorical move. "This is happening whether you like it or not, so get on board." It sounds like an argument, but it's actually a way to skip the argument, framing adoption as a foregone conclusion so you never have to defend it on the merits.

The honest version of that claim is much smaller: some LLM applications are genuinely useful right now, some are speculative, and some are actively worse than what they replace. Treating all three the same because "AI is inevitable" is how teams end up shipping features nobody asked for.

The useful discipline, especially for anyone advising others on where to apply this technology, is evaluating each use case on its own evidence rather than riding the narrative. Boring, but it's the only version of this that ages well.
