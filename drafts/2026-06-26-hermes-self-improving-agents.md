The most underrated problem in AI agents: they don't get better with use.

Every session starts from scratch. The agent has no memory of what worked last time, no accumulated shortcuts for your workflow, no way to improve from mistakes. Hermes Agent from Nous Research is trying to fix that with a built-in learning loop — the agent creates skills from experience, refines them during use, and persists them for future sessions.

The idea is simple: when the agent finds a good solution, it abstracts that into a reusable skill rather than discarding the insight when the context clears.

I don't think this is fully solved. The failure modes — bad skills accumulating alongside good ones — are real engineering challenges. But the direction is right. Agents that compound knowledge over time are fundamentally more useful than agents that reset. The gap between a fresh agent and one that has worked in your environment for a month should be enormous.
