## Themes

### Persistent Code Intelligence as an MCP Primitive

codebase-memory-mcp led GitHub trending at 2190 points, shipping a single static binary that indexes any codebase into a persistent knowledge graph in milliseconds across 158 languages with sub-millisecond queries and a claimed 99% token reduction. The design — zero dependencies, persistent state across sessions — positions code memory as infrastructure rather than a per-session prompt trick, and its top ranking suggests strong developer appetite for this abstraction.

Sources: <https://github.com/DeusData/codebase-memory-mcp>

### OpenAI's Internal Codex Metrics Signal Enterprise Inflection

OpenAI disclosed that internal median Codex output tokens have grown 56x in Research, 32x in Customer Support, 27x in Engineering, and 13x in Legal since November 2025 — embedding agentic coding across non-engineering functions at scale. On the same day, GPT-5.6 Sol/Terra/Luna launched in a restricted tiered rollout to trusted partners, mirroring Anthropic's own tiered release cadence; the simultaneous moves from both labs suggest coordinated competitive signaling around capability tiers.

### Agentic Finance Tooling Hits Saturation Point

Three independent multi-agent finance projects trended simultaneously: ai-berkshire (1445) applies Buffett/Munger/Duan methodologies via Claude Code with adversarial multi-agent analysis; Vibe-Trading (492) offers a one-command personal trading agent; TradingAgents v0.3.0 (274) ships a pluggable multi-LLM trading framework. The cluster suggests a gold-rush moment for this category, though none publicly report live portfolio validation — the real differentiation will come from risk management, not agent architecture.

Sources: <https://github.com/xbtlin/ai-berkshire> · <https://github.com/HKUDS/Vibe-Trading> · <https://github.com/TauricResearch/TradingAgents>

### Claude Code Expanding Into Non-Code Workflows

browser-use shipped video-use (196), enabling users to drop raw footage in a folder and produce a final cut purely through Claude Code chat — extending agentic coding into media production. A visual example-driven Claude Code guide (claude-howto, 312) also trended, reflecting that the tool is now far enough into mainstream adoption that structured onboarding material is a growth lever.

Sources: <https://github.com/browser-use/video-use> · <https://github.com/luongnv89/claude-howto>

### Document Ingestion Pipeline for LLM Workflows

MinerU (380) continues gaining traction as a tool that converts complex PDFs and Office documents into clean LLM-ready markdown or JSON, with zero-install web, desktop, and API access options. Its sustained GitHub presence reflects the unglamorous but persistent bottleneck in enterprise agentic workflows: getting structured data out of documents that were never designed for machines.

Sources: <https://github.com/opendatalab/MinerU>
