## Themes

### AI-Native Video Production Arrives

Two high-scoring releases on the same day signal a new category: AI-first video creation. OpenMontage (3.6k) ships as open-source with 12 pipelines, 52 tools, and 500+ agent skills, turning any AI coding assistant into a full video production system. Palmier Pro (1.6k) takes the native-app approach, a macOS video editor built specifically for AI workflows — notably requiring macOS 26 (Tahoe), Apple's not-yet-released OS, suggesting tight platform integration ahead.

Sources: <https://github.com/calesthio/OpenMontage> · <https://github.com/palmier-io/palmier-pro>

### Agent Skills Standard Coalesces

The agentskills.io standard is rapidly becoming the coordination layer for AI agent capabilities: Anthropic published its official skills repo, NVIDIA followed with verified agent skills tied to its docs and governance framework, and a 817-skill cybersecurity library mapped to MITRE ATT&CK and NIST surfaced for Claude Code and peers. Community tooling piled on — Garry Tan's gstack (1k), the ECC harness (593), and a meta-skill factory for generating domain-specific agent teams (revfactory/harness) — suggesting the ecosystem is moving from ad-hoc prompting to structured, reusable skill libraries.

Sources: <https://github.com/anthropics/skills> · <https://github.com/NVIDIA/skills> · <https://github.com/mukul975/Anthropic-Cybersecurity-Skills> · <https://github.com/garrytan/gstack> · <https://github.com/revfactory/harness> · <https://github.com/affaan-m/ECC>

### Persistent, Efficient Code Intelligence

codebase-memory-mcp (1.3k) ships a single static binary that indexes entire repos into a persistent knowledge graph in milliseconds across 158 languages, claiming 99% token reduction via sub-millisecond queries — a direct attack on the context-window tax agents pay when navigating large codebases. NousResearch's hermes-agent (936) complements this with a self-improving loop that creates and refines skills from experience. An arxiv paper on agent-native memory systems frames the theoretical stakes: memory is no longer retrieval-augmented lookup but a full data management lifecycle.

Sources: <https://github.com/DeusData/codebase-memory-mcp> · <https://github.com/NousResearch/hermes-agent> · <https://arxiv.org/abs/2606.24775v1>

### Long-Horizon Autonomous Research Agents

Karpathy's autoresearch (186) targets single-GPU nanochat training, framing fully automated AI research as imminent rather than speculative. ByteDance's deer-flow (739) takes the broader approach — an open-source SuperAgent harness for tasks taking minutes to hours, combining sandboxes, memory, subagents, and a message gateway. The OpenThoughts-Agent paper simultaneously surfaces public data recipes for training broadly capable agentic models, addressing the open-weights gap in multi-step reasoning agents.

Sources: <https://github.com/karpathy/autoresearch> · <https://github.com/bytedance/deer-flow> · <https://arxiv.org/abs/2606.24855v1>

### Local Voice I/O as a First-Class Stack

Voicebox (1k) ships a complete open-source voice studio — clone any voice, generate speech, dictate into any app, and talk to agents — running entirely locally. This isn't just a TTS wrapper: it's positioning the full voice I/O layer as something developers own and run on-device, a direct counter to locked-in cloud voice APIs. Combined with the growing agent ecosystem, local voice becomes another composable skill rather than a gated service.

Sources: <https://github.com/jamiepine/voicebox>
