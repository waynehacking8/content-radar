## Themes

### Agentic Output Is Already Scaling Dramatically

OpenAI's own internal Codex usage grew median output tokens 56x in Research, 32x in Customer Support, 27x in Engineering, and 13x in Legal since November 2025—concrete evidence that AI-assisted work is compounding at a pace few expected this soon. This is not projected growth: it's observed production throughput inside the organization that built the tool. The gap between 'AI helps a bit' and 'AI does most of the work' is closing faster than the public discourse reflects.

### Open-Source Agentic Pipelines Go Deep

OpenMontage (3.4k stars) debuted as the first open-source agentic video production system with 12 pipelines, 52 tools, and 500+ agent skills—turning a coding assistant into a full studio. Agent-Reach (1.5k stars) gives agents zero-API-fee access to Twitter, Reddit, YouTube, GitHub, and Chinese platforms via a single CLI. ByteDance's deer-flow adds a long-horizon SuperAgent harness for tasks spanning minutes to hours with sandboxes, memories, and subagents. Alibaba's page-agent lets natural language control arbitrary web interfaces in-browser. The common thread: teams are no longer building one-off agents but composable, multi-tool pipelines designed for real production workloads.

Sources: <https://github.com/calesthio/OpenMontage> · <https://github.com/Panniantong/Agent-Reach> · <https://github.com/bytedance/deer-flow> · <https://github.com/alibaba/page-agent>

### Claude Code Crystallizes as a Platform

Claude Code is behaving less like a tool and more like an OS: Garry Tan's gstack repo documents his full setup (23 opinionated tools covering CEO, Designer, Eng Manager, QA roles) with the claim he hasn't typed a line of code since December. A dedicated claude-code-best-practice repo formalizes the shift from 'vibe coding' to 'agentic engineering.' The AI website cloner template explicitly recommends Claude Code with Opus 4 as its primary runtime, and the ai-berkshire value-investing framework uses multi-agent Claude Code for adversarial research. Third-party workflows are now being built on top of Claude Code the way they once were built on top of bash.

Sources: <https://github.com/garrytan/gstack> · <https://github.com/shanraisshan/claude-code-best-practice> · <https://github.com/JCodesMore/ai-website-cloner-template> · <https://github.com/xbtlin/ai-berkshire>

### Infrastructure Layer for Agents Matures

Google Labs shipped DESIGN.md (1.5k stars), a format spec giving coding agents persistent, structured knowledge of a visual design system—the same declarative bet as CLAUDE.md but for brand/UI identity. AWS released an official Agent Toolkit with MCP servers and plugins for building and deploying on AWS infrastructure. Apple open-sourced a Swift-native container tool for running Linux containers as lightweight VMs on Apple Silicon. Together these signal the infrastructure layer solidifying: agents need standard interfaces for design context, cloud operations, and isolated execution environments.

Sources: <https://github.com/google-labs-code/design.md> · <https://github.com/aws/agent-toolkit-for-aws> · <https://github.com/apple/container>

### Agent Security Becomes a Real Discipline

NVIDIA released SkillSpector, a security scanner that detects vulnerabilities and malicious patterns in agent skills before installation—a direct response to the explosion of third-party agent skill registries. An arxiv paper on prompt injection in LLM resume screening shows candidates can already manipulate automated hiring systems in both single and multi-injection settings, which matters given the hiring-agent repo that appeared today (683 stars, resume-to-score pipeline). A separate dataset of 817 structured cybersecurity skills mapped to MITRE ATT&CK, NIST CSF, and D3FEND signals that defenders are systematically cataloguing what AI agents need to know. The attack surface of agentic systems is now drawing structured defensive research.

Sources: <https://github.com/NVIDIA/SkillSpector> · <https://arxiv.org/abs/2606.27287v1> · <https://github.com/mukul975/Anthropic-Cybersecurity-Skills> · <https://github.com/interviewstreet/hiring-agent>
